Patch Notes 01/12/2024

Optimising

Chapter Page

We now load the chapter you selected first, and then all of the other chapters, while all of the other chapters are still loading the next/previous chapter buttons will be disabled.

Improved Performance

Optimised our site to load faster and reduce server load.

Home Page

The home page no longer takes a few centuries to load. The base home page will be loaded and then it will fetch the stats from the server. (This is shown on the home page as the stats 'loading...')

Re-Directs

We have now added support for users who may have had a tab open before the swap to the new urls, this will now re-direct you to the correct url so you can continue reading where you left off if you forgot what chapter you where on.
